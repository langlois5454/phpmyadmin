Introduction

phpMyAdmin est une application web permettant de gérer des bases de données sur un serveur. En fait, sur le serveur tourne un serveur de base de données (comme MYSQL), et phpMyAdmin est une application PHP permettant de faire l'interface entre l'utilisateur et le serveur de bases de données.

Préparer son environnement

Avec son propre serveur local

Quand vous installez tout un paquet serveur web+php+MySQL, comme par exemple, WAMP, vous avez déjà à disposition l'outil phpMyAdmin. Normalement, l'installation a créé pour vous un utilisateur pour le serveur MySQL, et c'est cet utilisateur que vous allez utiliser pour vous connecter au serveur de base de données. En attendant plus d'informations sur quelques unes des nombreuses possibilités de pack, il faut vous reporter à la documentation du pack que vous avez choisi.

Il se peut que votre serveur local inclut un service équivalent à phpMyAdmin, comme adminer. C'est le cas du pack WinNMP. Si vous êtes dans ce cas, allez sur cette page qui donne la procédure pour adminer

Avec alwaysdata

Si vous utilisez l'hébergeur alwaysdata, il faut d'abord créer un utilisateur de bases de données, puis une base de données vide dans laquelle vous allez importer les données de world.sql.

Voici comment faire :

  • Connectez-vous sur votre compte alwaysdata
  • Cliquez sur la gauche sur Bases de données/MySQL.
  • Création de l'utilisateur
    • Cliquez en haut sur Utilisateurs
    • Cliquez sur Ajouter un utilisateur (à droite)
    • Donnez un nom à l'utilisateur (il se peut qu'un préfixe vous soit imposé), choisissez un mot de passe
    • Cliquez sur Valider
  • Création de la base de données
    • Cliquez maintenant sur Bases de Données, en haut
    • Cliquez sur Ajouter une base de données
    • Donnez un nom à la base de données, peut-être qu'un préfixe vous sera imposé
    • Donnez tous les droits à l'utilisateur que vous avez créé
    • Cliquez sur Valider

Ca y est, votre environnement est prêt. Vous allez maintenant utiliser phpMyAdmin pour importer le fichier world.sql dans votre base de données toute neuve (mais vide).

Importer une base de données avec phpMyAdmin

Il faut d'abord accéder à phpMyAdmin. Pour ce faire, cela dépend de votre serveur.

Si vous avez un serveur local, il existe normalement une page d'accueil de votre serveur, page qui contient un lien vers phpMyAdmin.

Si vous utilisez alwaysdata, à partir de Bases de données/MySQL, il y a un lien en haut phpMyAdmin. Vous devrez donner comme identifiant et mot de passe les informations associées à l'utilisateur que vous avez créé dans la section précédente.

Maintenant, pour importer le fichier world.sql, voici comment faire :

  • Préparer le fichier worldutf8.sql, disponible sur le cours en ligne. En fait, au tout début du cours, ce fichier s'appelait world.sql. Mais certains d'entre-vous se sont rendus compte d'un problème d'encodage. David Grosdemange a proposé une nouvelle version qui corrige ce problème. C'est cette version worldutf8.sql qui est disponible en ligne.
  • Connectez-vous sur le service phpMyAdmin
  • Attention important : cliquez sur la base (vide) que vous avez créée précédemment, afin de la sélectionner pour y importer le fichier wordlutf8.sql. Je note "Important", car personnellement, j'oublie systématiquement de le faire
  • Cliquez sur Import dans les onglets du haut
  • Cliquez sur Parcourir, et allez chercher le fichier worldutf8.sql
  • Sélectionnez utf-8 dans la liste Character set of the file:.
  • Cliquez sur GO

Si tout s'est bien passé, au bout de quelques dizaines de secondes, vous voyez apparaître, sur la gauche, dans votre base de données, les trois tables City, Country, et CountryLanguage.

Vous pouvez maintenant vous amuser avec cette base via phpMyAdmin. Pour ce faire, vous pouvez utiliser les ressources diffusées par Marie.

Vous pouvez aussi revenir au document de cours (section Bases de données relationnelles et PHP) afin de voir comment interroger la base directement avec PHP. Mais pour cela, il faut bien mémoriser le nom du serveur MySQL et le nom de la base, donnés en haut de l'interface de phpMyAdmin.
